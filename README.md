# Privacy-Preserving Whistleblower System using ZK-SNARKs

> **Prove a classified document contains sensitive information without revealing the file, your identity, or your location.**

A web-based cryptographic system that allows a whistleblower to prove that a file was digitally signed by Company-X and contains the literal phrase `"Confidential: Toxic Waste"` — using zero-knowledge proofs (Groth16 ZK-SNARKs).

**Authors:** Abhilash Purohit, Suchet Kumbar, Pratik Anand  
**Course:** Computer Network Security (CNS)

## How to Run in 5 Commands

```bash
# 1. Install all dependencies
npm install && cd web/server && npm install && cd ../client && npm install && cd ../..

# 2. Build the circuit (WSL2 required)
bash scripts/buildCircuit.sh

# 3. Run the full demo pipeline (chunk → sign → prove → verify)
npm run demo

# 4. Start the verifier server (new terminal)
npm run server

# 5. Start the web app (new terminal)
npm run client
# Open http://localhost:3000 and upload sample_doc.txt
```

## System Architecture

```
Company-X (Offline)              Whistleblower (Browser)           Verifier (Server)
─────────────────────           ────────────────────────           ─────────────────
1. Hash file → chunks            4. Read file locally               8. Receive proof only
2. Build Poseidon Merkle tree    5. Find chunk with phrase           9. Registry check
3. EdDSA sign Merkle root        6. Build Merkle inclusion proof    10. Groth16 verify
                                 7. Generate ZK proof (WASM)        11. Accept / Reject
                                    Private: [chunk, path, sig]
                                    Public:  [root, phrase]
```

### What the Proof Proves
1. ✅ The prover knows a document chunk containing the target phrase
2. ✅ The chunk is correctly hashed via Poseidon
3. ✅ The chunk's hash is a leaf in the Merkle tree with the given root
4. ✅ (Optional) The root was signed by Company-X's EdDSA key

### What is NOT Revealed
- ❌ The document's actual content
- ❌ Which chunk contains the phrase
- ❌ The whistleblower's identity or IP
- ❌ The Merkle path (sibling hashes)

## Technologies

| Component | Technology | Why |
|-----------|-----------|-----|
| **Proving System** | Groth16 (snarkjs) | Smallest proof size (192B), fastest verification |
| **Circuit Language** | Circom v2 | Industry standard, great circomlib ecosystem |
| **Hash Function** | Poseidon | ZK-friendly (~8x fewer constraints than SHA-256) |
| **Signature** | EdDSA (Baby Jubjub) | ZK-native curve, ~5K constraints vs ~500K for ECDSA |
| **Merkle Tree** | Poseidon-based binary tree | Proves chunk membership without revealing other chunks |
| **Frontend** | Vite + React 18 | Modern, fast HMR, native ESM |
| **Backend** | Express.js | Verify-only server, no document uploads |
| **3D Background** | Three.js + R3F | Premium UI aesthetic |

## Cryptographic Details

| Parameter | Value |
|-----------|-------|
| Chunk Size | 64 bytes (configurable) |
| Phrase Length | 25 bytes ("Confidential: Toxic Waste") |
| Merkle Tree Depth | 10 (supports up to 1024 chunks = 64KB) |
| Proof Size | ~192 bytes |
| Verification Time | < 10ms |
| Proving Time | ~5-15s (browser, depends on circuit size) |
| Constraint Count | ~5,600 (demo) / ~15,600 (with EdDSA) |
| PTAU Required | pot15 (2^15 = 32,768 max constraints) |

## Debugging Tips

### Witness Generation Fails
```bash
# Inspect the witness
snarkjs wtns debug build/whistleblower.r1cs input/input.json build/whistleblower.sym

# Common causes:
# - Phrase not found in chunk (matcher.found !== 1)
# - Merkle root mismatch (computed root ≠ public root)
# - Path indices incorrect (0=left, 1=right)
```

### Circuit Compilation Errors
```bash
# Check constraint count before setup
snarkjs r1cs info build/whistleblower.r1cs

# If circuit is too large, reduce:
# - Merkle depth (10 → 5)
# - Chunk size (64 → 32)
# - Disable EdDSA (set ENABLE_EDDSA=0)
```

### Proof Verification Fails
```bash
# Verify locally first
node scripts/verifyProof.js

# Check that verification_key.json matches circuit_final.zkey
snarkjs zkey verify build/whistleblower.r1cs pot15_final.ptau build/circuit_final.zkey
```
